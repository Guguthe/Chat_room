# python聊天室
一个基于python的聊天室，使用pyside6作为gui驱动程序，你可以通过配置mysql数据库来实现聊天室的登陆操作

## 功能
- 可以进行用户的注册与登陆
- 消息实时实时传输
- 可以在linux or windows 平台上使用

## 使用流程
1. 通过git或下载获取项目文件
2. 运行server.py作为服务端，main作为客户端
3. 在登陆后进行收发消息
